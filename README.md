# zkcli - 一个zookeeper客户端工具

## 使用

使用前需要先初始化zk配置：

```bash
$ zkcli init 192.168.1.1:2181,192.168.1.2:2181,192.168.1.3:2181
```

以树形结构列出子节点信息：
```bash
$ zkcli tree /foo
/foo
├── bar
│   ├── a
│   ├── b
│   └── c
└── languages
    ├── php
    ├── java
    └── golang
```

删掉节点及其所有子节点：
```bash
$ zkcli rm -r /foo
Deleted /foo/bar/a
Deleted /foo/bar/b
Deleted /foo/bar/c
Deleted /foo/bar
Deleted /foo/languages/php
Deleted /foo/languages/java
Deleted /foo/languages/golang
Deleted /foo/languages
Deleted /foo
```
